2. Добавлены два новых класса (+третий привязка к первому)
    ContainerList (+Item) — работа со всеми функциями контейнера-списка
    ContainerArray — работа со всеми функциями контейнера-массива
    Коды контейнеров при работе:
    ContainerList -> buttonKey == 23
    ContainerArray -> buttonKey == 23
3. Список всех функций для элементов контейнеров-списков и конт-массивов с их actionKey
    Создать(1) — переместить(2) — показать(3) — стереть(4) — уничтожить(5) - добавить(6)
4. Список всех функций в main используемых в этой ЛР: //стар == дополнение к предыдущим ЛР
    Работа с панелью кнопок:
    -> back() — логика работы кнопки назад (стар)
    -> createPanel() — обновление или первичное создание панели с кнопками (стар)
    Работа с фигурами на холсте
    -> createRandom() — заполнение контейнера фигурами (стар)
    -> addFigure() — добавление фигуры в контейнер
    -> moveToObject() — перемещение фигуры на холсте(стар)
    -> deleteAll() — уничтожение контейнера (стар)
    -> eraseAll() — стирание объектов с холста (стар)
    -> showAll() — рисование объектов на холсте (стар)
    -> changeSize() — изменение размеров ЧЕТЫРЕХУГОЛЬНИКОВ (стар)
    -> changeRadius() — изменение радиусов ОКРУЖНОСТЕЙ (стар)
